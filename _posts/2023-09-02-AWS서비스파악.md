### 분석

- Amazon Athena
```
- Amazon Athena는 표준 SQL을 사용하여 Amazon S3의 데이터를 쉽게 분석할 수 있게 해주는 대화형 쿼리 서비스입니다. Athena는 서버리스이므로 설정하거나 관리할 인프라가 없으며 실행한 쿼리에 대해서만 비용을 지불합니다. 시작하려면 S3의 데이터를 가리키고 스키마를 정의한 다음 표준 SQL을 사용하여 쿼리를 시작하면 됩니다.
```
- AWS Data Exchange
```
- AWS Data Exchange는 고객이 AWS 클라우드에서 타사 데이터를 쉽게 찾고, 구독하고, 사용할 수 있게 해주는 서비스입니다.
```
- AWS Data Pipeline
```
- AWS Data Pipeline은 데이터 이동 및 변환을 자동화하는 데 사용할 수 있는 웹 서비스입니다. AWS Data Pipeline을 사용하면 데이터 기반 워크플로를 정의할 수 있으므로 이전 작업의 성공적인 완료에 따라 작업이 달라질 수 있습니다.
```
- Amazon EMR
```
- Amazon EMR은 Apache Hadoop과 Amazon Web Services에서 제공하는 서비스를 사용하여 방대한 양의 데이터를 쉽고 효율적으로 처리할 수 있게 해주는 웹 서비스입니다.
```
- AWS Glue
```
- AWS Glue는 분석, 기계 학습 및 애플리케이션 개발을 위해 데이터를 쉽게 검색, 준비 및 결합할 수 있게 해주는 확장 가능한 서버리스 데이터 통합 ​​서비스입니다.
```
- Amazon Kinesis
```
- Amazon Kinesis를 사용하면 비디오 및 데이터 스트림을 실시간으로 쉽게 수집, 처리 및 분석할 수 있습니다.
```
- AWS Lake Formation
```
- AWS Lake Formation은 데이터 레이크를 쉽게 설정, 보호 및 관리할 수 있게 해주는 관리형 서비스입니다. Lake Formation은 데이터 원본을 검색한 다음 데이터를 분류, 정리 및 변환하는 데 도움이 됩니다. Lake Formation을 사용하여 데이터를 보호하고 Amazon Simple Storage Service(Amazon S3) 데이터 레이크로 수집할 수 있습니다.
```
- Amazon Managed Streaming for Apache kafka(Amazon MSK)
```
- Amazon Managed Streaming for Apache Kafka(Amazon MSK)는 Apache Kafka를 사용하여 스트리밍 데이터를 처리하는 애플리케이션을 쉽게 구축하고 실행할 수 있게 해주는 완전관리형 서비스입니다.
```
- Amazon OpenSearch Service(Amazon Elasticsearch Service)
```
- Amazon OpenSearch Service는 인기 있는 오픈 소스 검색 및 분석 엔진인 OpenSearch를 쉽게 배포, 운영 및 확장할 수 있게 해주는 관리형 서비스입니다. OpenSearch 서비스는 또한 보안 옵션, 고가용성, 데이터 내구성 및 OpenSearch API에 대한 직접 액세스를 제공합니다.
```
- Amazon QuickSight
```
- Amazon QuickSight는 시각화를 구축하고, 임시 분석을 수행하고, 데이터에서 신속하게 비즈니스 통찰력을 얻을 수 있는 빠른 비즈니스 분석 서비스입니다. Amazon QuickSight는 AWS 데이터 소스를 원활하게 검색하고, 조직이 수십만 명의 사용자로 확장할 수 있도록 지원하며, Amazon QuickSight SPICE(초고속, 병렬, 인 메모리, 계산 엔진)를 사용하여 빠르고 응답성이 뛰어난 쿼리 성능을 제공합니다.
```
- Amazon Redshift
```
- Amazon Redshift는 기존 비즈니스 인텔리전스 도구를 사용하여 모든 데이터를 간단하고 비용 효율적으로 효율적으로 분석할 수 있게 해주는 빠르고 완벽하게 관리되는 페타바이트 규모의 데이터 웨어하우스 서비스입니다. 이는 수백 기가바이트에서 페타바이트 이상의 데이터 세트에 최적화되어 있으며 비용은 연간 테라바이트당 $1,000 미만입니다. 이는 대부분의 기존 데이터 웨어하우징 솔루션 비용의 10분의 1입니다.
```
